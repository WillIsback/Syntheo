Architecture stack
Julie P. 04:22
Donc côté technique, on reste sur le VPS OVH. Tout tourne en Docker Swarm avec un rolling release pour le zéro downtime. WhisperX et vLLM sont dans des containers isolés, sans aucun port exposé à l'extérieur. Le frontend Next.js est le seul point d'entrée, via Traefik en reverse proxy avec TLS 1.3.

Ce qui me rassure c'est l'absence de conservation audio. C'est la décision la plus déterminante pour le périmètre de l'AIPD.
